Cricket Mania

Use the Native5 API Client to fetch data from ESPN's Cricket news and display cool stats using Native5 UI Widgets

  • ESPN
    • Register at ESPN Developer Website to get your API Key [http://developer.espn.com/docs]
    • Get cricket headlines:  http://api.espn.com/v1/sports/cricket?apikey=<You API Key>
  • Native5 UI Widgets [http://docs.native5.com/repository/current/sdk-ui/js/modules/ui.html]
  • Native5 PHP Framework [http://docs.native5.com/repository/current/sdk-php/native5-sdk-client-php/]
  • Links:
    • Native5 Developer Documentation: http://docs.native5.com
    • Native5 Website: http://native5.com
