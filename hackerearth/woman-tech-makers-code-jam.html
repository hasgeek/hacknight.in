Woman Tech Makers Code Jam

The organisers of GDG Delhi suggested that they have a Hackathon hosted on HackerEarth,the women tech makers code jam, and they would invite women programmers to participate in the Hackathon at an assigned location. This would happen across India, wherever feasible through the various GDG chapters. There are awesome prizes to be won, so if there’s a GDG organising a woman only Hackathon on the 8th of march, make sure you’re there!

We at HackerEarth want to take this a step further. If you’re a woman programmer, anywhere in the world, you can participate in the women tech makers code jam Hackathon. There will be a separate leaderboard for programmers who weren’t at a GDG organised event and we’re giving away awesome music gear from Apple for the top 2 winners!

More about the event here - http://blog.hackerearth.com/2014/03/a-woman-only-hackathon.html
  • Sat, 8 Mar 3:30 PM — 6:00 PM 8 Mar 2014
  • Entry Fee: Free of charge
  • Centre for Internet and Society
  • No 194, 2nd C Cross, 4th Main
  • Opp. Domlur Club, Domlur 2nd Stage
  • Bangalore, Karnataka – 560 071, IN
